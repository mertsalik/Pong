Pong
====

A GLSL workout on classical Pong game.


----------
#### Requires ####

- Open Asset Import Library ([Assimp v3.0](http://assimp.sourceforge.net/main_downloads.html))

- OpenGL Mathematics ([GLM 0.9.4.3](http://sourceforge.net/projects/ogl-math/))

- MS Visual Studio 2012 | 2010

	- ProjectDefaults->Character Set : NOT SET

	- Linker->Input += assimp.lib;

	- Include Directories += GLM & Assimp include paths;

	- Library Directories += Assimp lib path;
